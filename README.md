# PictureSelector
这是一个仿微信的图片选择器：

1、图片来源：拍照，本地jpg和png类型的图片

2、功能：多选，单选，图片预览，裁剪

3、todo：使用MVP模式重构代码、优化细节（一些动画）、整理解析文档
